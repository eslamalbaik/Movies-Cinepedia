
## Run Locally

Clone the Repository

```bash
git clone https://github.com/eslamalbaik/Movies-Cinepedia.git
```

environment variables

```bash
VITE_APP_MOVIE_API = YOUR_TMDB_API_ACCESS_TOKEN
```

Installation

```bash
npm install
```

Start the server
```
npm run dev  
```


## Tech Stack Used

***React, HTML, SCSS, Vite.js***
